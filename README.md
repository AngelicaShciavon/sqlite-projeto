# sqlite-projeto

Estudos e exercícios práticos de **SQL com SQLite**.

Organizado por seções para facilitar o aprendizado:

- **secao04**: Criação de tabelas (DDL), inserções/atualizações/exclusões (DML), transações (BEGIN/COMMIT/ROLLBACK)
- **secao05**: Filtros (WHERE), junções (JOINs), agregações (COUNT, MAX, AVG, etc.), ordenação (ORDER BY)
- **juncao**: Exemplos variados de joins (INNER, LEFT, SELF JOIN, produto cartesiano)
- **agregacao**: GROUP BY + HAVING + funções de agregação em cenários reais (ex: estoque por fabricante)

## Como usar
1. Instale o SQLite (ou use DB Browser for SQLite)
2. Execute os arquivos .sql na ordem: ddl → dml → consultas
3. Os arquivos .db são os bancos gerados (não versionados no Git)
